Prompt GO

คำอธิบาย

Prompt GO เป็นแอปสำหรับเรียกรถไฟฟ้าภายในมหาวิทยาลัย ออกแบบมาเพื่อเพิ่มความสะดวกในการเดินทางของนักศึกษาและบุคลากร ผู้ใช้งานสามารถดูตำแหน่งรถและสถานีได้แบบเรียลไทม์ เลือกจุดขึ้นและลงเพื่อประเมินเวลาและระยะทางโดยประมาณ พร้อมทั้งกดเรียกรถและติดตามสถานะการเดินทางได้ทันที ระบบยังส่งการแจ้งเตือนเมื่อรถใกล้ถึงสถานีที่กำหนด ช่วยลดเวลารอและทำให้การเดินทางภายในมหาวิทยาลัยมีประสิทธิภาพมากขึ้น

หน้าที่ที่รับผิดชอบ (Tester)

ดำเนินการ Manual Testing: ออกแบบ Test Case, ทดสอบการทำงานของแอป และบันทึกผลลง Google Sheet พร้อมสร้างตารางสรุปผลการทดสอบเพื่อใช้ติดตามปัญหา พัฒนาและรัน Automated Test ด้วย Robot Framework ใช้ Appium Library เชื่อมกับ Android Studio Emulator เพื่อทดสอบฟังก์ชันซ้ำ ๆ ในแอปพลิเคชันมือถือ ใช้ Selenium Library สำหรับทดสอบฟังก์ชันของฝั่ง Web Application ทำ Security Testing (Pentest เบื้องต้น) ตรวจสอบช่องโหว่ Cross-Site Scripting (XSS) Stored และจัดทำรายงานผลการทดสอบส่งให้ทีมพัฒนา (Development Team)
